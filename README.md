A Live Enquiry Management System is a web-based application that allows users (customers or students) to submit enquiries in real time, while administrators can view, manage, and respond to those enquiries efficiently.


# Live Enquiry Management System

## Description
A web-based application that allows users to submit enquiries online and enables administrators to manage and respond to them efficiently. This project follows industry-level standards suitable for IBM enterprise applications.

## Features
- User enquiry submission
- Admin dashboard
- Enquiry status tracking
- Secure authentication
- MongoDB database integration

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## How to Run
1. Clone the repository
2. Run npm install
3. Start server using node server.js
